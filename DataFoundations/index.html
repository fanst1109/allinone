Lĩnh vực này dạy gì?

  • Cách máy biểu diễn số: binary/hex, two's complement, IEEE 754 floating-point.

  • Math foundations: set theory, boolean logic, combinatorics, modular arithmetic, proof & induction.

Cần khi học DataStructures (bitmask, hash, Bloom), Cryptography (XOR, modular, hash), Databases (mô hình quan hệ = set theory, encoding), AI-ML (numerical precision).
