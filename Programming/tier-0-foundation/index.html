Tier 0 — Foundation / Mindset

5 lesson nền tảng — chưa chạm code Go, tập trung vào tư duy và môi trường làm việc.

Tier này dạy gì?

Trước khi học bất kỳ ngôn ngữ nào, bạn cần biết: cách suy nghĩ như lập trình viên, dùng môi trường dev và Git, chạy được command line, đọc code lạ và debug. Tier 0 lấp đầy phần "ngầm" mà các giáo trình ngôn ngữ thường giả định bạn đã biết. Lesson 05 mới chạm tới Go — giới thiệu triết lý và lý do tồn tại.

5 lesson trong Tier 0
Lesson 01
Tư duy lập trình
Decompose vấn đề, pattern matching, mental model, viết pseudocode trước khi viết code.

Môi trường dev & Git
Setup OS, editor, terminal, Git workflow (clone/branch/merge/rebase) — kỹ năng nền cho mọi project.

Lesson 03
Command Line
bash/zsh, pipe, redirect, find/grep/sed/awk, ssh, tmux — công cụ năng suất số 1 của dev.

Lesson 04
Đọc code & Debug
Đọc codebase lạ, đọc spec / stack trace, debug bằng print và debugger.

Lesson 05
Vì sao Go?
Lịch sử Go, triết lý "simplicity over cleverness", so sánh với C/Java/Python/Rust. Hello world đầu tiên.
