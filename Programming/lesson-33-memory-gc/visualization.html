Module 2 — Tri-color Mark & Sweep

    

      12 object node + 1 root. Bấm Step để chạy mark phase từng bước theo BFS từ
      root. Mỗi bước: pop 1 grey object, push các neighbour reachable thành grey, mark object đó
      black. Cuối cùng: mọi node còn white = garbage.

Module 3 — GOGC Tuner

    

      Kéo slider GOGC từ 50 → 400. Chart mô phỏng heap grow theo thời gian: mỗi lần đạt
      heap_live × (1 + GOGC/100) thì GC chạy, drop heap về live.
      Quan sát trade-off: GOGC thấp = nhiều GC, pause/lần ngắn, peak heap thấp; GOGC cao =
      ít GC, pause dài, peak heap cao.
