Tầng 6 — AI/ML

Ứng dụng: ráp tất cả nền toán Tầng 1-5 thành ML thật — từ linear regression tới CLIP.

Tầng này dạy gì?

Tầng cuối, đích đến của lộ trình. Mọi thứ học ở 5 tầng trước (vector, gradient, chain rule, MLE, cross-entropy) đều được dùng đến. Từ linear regression đơn giản tới CLIP multimodal — cùng một bộ machinery.

  • Pipeline ML end-to-end: data → feature → model → loss → train → evaluate.
  • Linear/logistic regression, neural network 1-hidden-layer.
  • Text vectorization (BoW, TF-IDF) → word embedding (Word2Vec).
  • Vector DB + RAG pipeline, CLIP multimodal.
