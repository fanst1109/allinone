AVL Tree — Cân bằng tự động

4 case rotation:
Số rotation đã thực hiện: 0 · Số node: 0 · Chiều cao: 0
Chèn từng số. Nếu cân bằng vượt ±1 → tự động rotation.

Số nhỏ bên dưới node: balance factor = height(left) - height(right). Node ĐỎ: balance vượt ±1 trước khi rotation. Node VÀNG: sau rotation.
