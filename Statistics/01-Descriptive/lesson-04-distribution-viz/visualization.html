Lesson 04: Trực quan hoá phân phối

Histogram, boxplot, violin, Q-Q plot, ECDF — chọn chart đúng cho mục đích. Cùng dataset, xem qua 4 biểu đồ khác nhau.

Module 1: Histogram — thay đổi bin width

Chọn phân phối, kích cỡ mẫu, rồi kéo slider bin width. Quá ít bin → mất chi tiết. Quá nhiều → nhiễu.

15
Quy tắc gợi ý sẽ hiển thị ở đây.

Module 2: Boxplot — Q1, Q3, IQR, outlier 1.5×IQR

Cùng dataset của Module 1. Xem 5-number summary và quy tắc whisker 1.5×IQR.

Module 3: Q-Q plot — dữ liệu có Normal không?

Q-Q plot vẽ quantile mẫu chống lại quantile lý thuyết Normal. Nếu dữ liệu Normal → các điểm nằm trên đường thẳng y=x. Nếu lệch → cong.

Module 4: ECDF — hàm phân phối tích lũy thực nghiệm

ECDF không cần chọn bin. Mỗi điểm dữ liệu tạo bậc thang nhảy 1/n. Đọc: F(x) = "tỉ lệ điểm ≤ x".
