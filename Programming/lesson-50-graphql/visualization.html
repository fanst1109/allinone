Ba mô-đun tương tác: (1) GraphQL vs REST — 1 query thay N request; (2) N+1 → DataLoader — gom ID batch 1 query; (3) Query depth limit — chặn query đệ quy sâu.

1. GraphQL vs REST — 1 query thay N request

Cùng một màn hình cần: user + 3 post + mỗi post 2 comment. REST phải đi nhiều round-trip; GraphQL gom thành 1 query duy nhất. Bấm "Fetch" để xem.

3. Query depth limit — chống DoS

Nhập query (kiểu rút gọn, dùng { } lồng nhau). Trang tính depth = số cấp lồng sâu nhất, so với limit. Query đệ quy friends { friends { ... } } sẽ vượt limit và bị reject.
