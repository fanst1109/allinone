Tầng 1 — Foundations

Nền tảng & quy trình — cách làm phần mềm cùng nhóm: vòng đời, Agile, yêu cầu, git, review, ước lượng.

Tầng này dạy gì?

Từ chỗ chỉ biết viết code một mình tới chỗ tham gia một nhóm kỹ thuật: hiểu vòng đời phần mềm, làm việc theo Agile, biến yêu cầu mơ hồ thành đặc tả rõ, cộng tác qua git & code review, và ước lượng công việc.

  • 6 bài, học tuần tự — bài sau dùng kiến thức bài trước.
  • Mỗi bài có README (lý thuyết + bài tập + lời giải) và visualization.html (minh họa tương tác).

▶ Visualization 📖 README

Functional vs non-functional, user story, acceptance criteria, đặc điểm yêu cầu tốt.

Lesson 06
Ước lượng & quản lý công việc
Story point, planning poker, velocity & dự báo, đọc burndown chart.
