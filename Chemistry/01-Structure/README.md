# Tầng 1 — Structure (Cấu tạo chất)

Từ "nguyên tử là gì" tới "phân tử hình thành thế nào và phản ứng ra sao". Sau tầng này, bạn hiểu cấu tạo chất và phân tích được phản ứng đơn giản — nền cho mọi bài hóa sau.

## Mục tiêu tổng quát

- Mô tả nguyên tử bằng proton/neutron/electron; đọc được số khối, đồng vị.
- Viết cấu hình electron và đọc bảng tuần hoàn — hiểu xu hướng tuần hoàn.
- Phân biệt liên kết ion, cộng hóa trị (cực, không cực); hiểu liên kết hydrogen.
- Dự đoán hình học phân tử bằng VSEPR.
- Cân bằng phương trình phản ứng và làm bài toán mol thành thạo.
- Tính nồng độ dung dịch và pha loãng.
- Hiểu tốc độ phản ứng, cân bằng hóa học, nguyên lý Le Chatelier.
