Lesson 07 — Biến & Kiểu dữ liệu trong Go

Ba module tương tác: (1) explore zero value & byte size của các kiểu Go, (2) thấy overflow wrap around bằng binary, (3) thử công thức iota live.

1. Zero Value Explorer

Chọn một kiểu, xem zero value và kích thước byte (trên platform 64-bit thông thường).

Khai báo:
var x int
Zero value:
0
Byte size:
8 (trên 64-bit)
So sánh với nil:
— (kiểu non-nullable)
int là kiểu integer mặc định trong Go. Trên máy 64-bit, kích thước 8 byte = int64; trên 32-bit là 4 byte = int32. Zero value luôn là 0.
