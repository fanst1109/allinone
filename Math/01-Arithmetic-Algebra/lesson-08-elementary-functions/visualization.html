Lesson 08 — Hàm sơ cấp

So sánh tăng trưởng: log < tuyến tính < bình phương < mũ

Đồ thị 4 hàm cùng lúc. Quan sát: khi x lớn, hàm mũ vượt xa tất cả.

Quan sát:

  • x = 1: tất cả gần nhau.
  • x = 5: x = 5, x² = 25, 2⁵ = 32, log₂(5) ≈ 2.3.
  • x = 10: x = 10, x² = 100, 2¹⁰ = 1024, log₂(10) ≈ 3.3.
  • Hàm mũ vượt hàm bình phương ở x ~ 4. Trong CS, đó là tại sao thuật toán O(2ⁿ) không dùng được với n lớn.

Hàm bậc 1 — Đường thẳng y = ax + b

a = 2 b = -3

Hàm bậc 2 — Parabol y = ax² + bx + c

a = 1 b = -4 c = 3
