4. User Mode vs Kernel Mode — So sánh

👤 User Mode (Ring 3)
  • Ứng dụng người dùng
  • Không truy cập I/O trực tiếp
  • Không đọc/ghi RAM tuỳ ý
  • Không tắt ngắt (interrupt)
  • Lệnh vi phạm → CPU exception
  • Ví dụ: Chrome, Python, bash
⚙ Kernel Mode (Ring 0)
  • OS Kernel
  • Truy cập mọi thiết bị I/O
  • Đọc/ghi bất kỳ địa chỉ RAM
  • Lập trình bảng trang (MMU)
  • Bật/tắt interrupt
  • Ví dụ: Linux kernel, NTFS driver
Cánh cửa duy nhất: System Call (SYSCALL instruction) User Mode muốn làm điều đặc quyền → gọi syscall → CPU trap → Kernel xử lý → SYSRET về User Mode
