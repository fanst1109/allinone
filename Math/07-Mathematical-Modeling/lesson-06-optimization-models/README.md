# Lesson 06 — Mô hình tối ưu (LP, Lagrange)

> 🚧 **Khung** — folder và mục tiêu đã có; nội dung chi tiết + visualization sẽ bổ sung sau.

## Mục tiêu (dự kiến)

- Mô hình bài toán "làm sao tốt nhất dưới ràng buộc": tối đa lợi nhuận, tối thiểu chi phí.
- **Quy hoạch tuyến tính (Linear Programming)**: hàm mục tiêu + ràng buộc tuyến tính; miền khả thi là đa giác lồi; nghiệm tối ưu ở đỉnh; giải bằng phương pháp đồ thị và ý tưởng đơn hình (simplex).
- **Tối ưu có ràng buộc đẳng thức — nhân tử Lagrange**: ∇f = λ∇g; diễn giải λ là "giá bóng" (shadow price).
- Liên hệ cực trị hàm nhiều biến ([T6 L04](../../06-Advanced/lesson-04-multivariable-functions/)).
