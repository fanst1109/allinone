# Tầng 1 — Foundations (Nền tảng & quy trình)

Tầng mở đầu của lĩnh vực [SoftwareEngineering](../README.md). Mục tiêu: từ chỗ chỉ viết code một mình, bạn tham gia được vào một nhóm kỹ thuật — hiểu vòng đời phần mềm, làm việc theo Agile, biến yêu cầu mơ hồ thành đặc tả rõ, cộng tác qua git & code review, và ước lượng công việc.

## Tầng này dạy gì?

- Vòng đời phần mềm (SDLC) và vì sao "code chạy được" chưa đủ.
- Quy trình lặp: Agile, Scrum (sprint/backlog), Kanban (giới hạn WIP).
- Thu thập & đặc tả yêu cầu: functional/non-functional, user story, acceptance criteria.
- Cộng tác qua git: feature branch, GitFlow vs trunk-based, merge conflict.
- Code review: pull request, checklist, văn hóa feedback.
- Ước lượng: story point, planning poker, velocity, burndown.
