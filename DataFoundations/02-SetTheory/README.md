# Nhóm 2 — Lý thuyết tập hợp — Lý thuyết tập hợp

Tập hợp, phép toán trên tập, ánh xạ và quan hệ tương đương — nền cho HashSet, HashMap, Union-Find và biểu diễn đồ thị.

## Mục tiêu

- Nắm các phép toán tập hợp: hợp, giao, hiệu, phần bù.
- Hiểu ánh xạ (mapping) — liên hệ trực tiếp tới HashMap.
- Hiểu quan hệ tương đương — liên hệ tới Union-Find.
