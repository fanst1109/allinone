Lesson 12 — Tối ưu hóa (Optimization)

Ba module tương tác trên IR ba địa chỉ: (1) pipeline bật/tắt từng pass tối ưu, (2) constant-folding stepper gập biểu thức từng bước, (3) dead-code highlighter tô lệnh sẽ bị bỏ. Mọi pass viết bằng JS thật — bạn sửa input là thấy kết quả đổi theo.

1. Optimization Pipeline — bật/tắt từng pass

Nhập IR ba địa chỉ, mỗi dòng một lệnh dạng x = a op b, x = a, hoặc return x. Toán tử: + - * /. Bật/tắt các pass để xem IR rút gọn dần và số lệnh trước/sau.
