SQL cơ bản — Playground giả lập

Một "SQL playground" chạy hoàn toàn bằng JavaScript trên bảng products mẫu (8 dòng). Không cần kết nối database thật. Bấm truy vấn dựng sẵn hoặc tự ghép mệnh đề để xem kết quả.

Bảng nguồn: products (8 dòng)

Mọi truy vấn dưới đây chạy trên đúng bảng này. Dòng id=8 có stock là NULL.

1 · Truy vấn dựng sẵn

Mỗi nút gắn 1 truy vấn. Bấm để chạy — kết quả hiển thị ở bảng bên dưới, kèm số dòng.

2 · Tự ghép truy vấn

Chọn cột, điều kiện WHERE, sắp xếp và giới hạn. Câu SQL tương ứng được sinh ra và thực thi ngay.

3 · Thứ tự thực thi: FROM → WHERE → SELECT → ORDER BY → LIMIT

Minh họa truy vấn SELECT name, price FROM products WHERE price > 100 ORDER BY price DESC LIMIT 3. Bấm "Chạy từng bước" để xem dây chuyền.
