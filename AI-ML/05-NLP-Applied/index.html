Tier 5 — NLP & Multimodal Applications

Ứng dụng cuối: text vectorization → word embeddings → vector DB + RAG → CLIP multimodal. Khép pipeline NLP & LLM hiện đại.

Tier này dạy gì?

  • Text vectorization cổ điển: one-hot, Bag-of-Words, TF-IDF.
  • Word embeddings: Word2Vec (skip-gram, CBOW), GloVe; vector arithmetic.
  • Vector DB + RAG: HNSW, FAISS, retrieve-augment-generate pipeline.
  • CLIP: ảnh + text cùng không gian, contrastive learning.
