Lesson 49 — gRPC & Protocol Buffers

Proto schema → wire format · 4 loại RPC · gRPC vs REST hiệu năng.

1. Proto schema → wire format

Hover (hoặc bấm) một field trong file .proto để xem nó encode thành bytes ra sao. Wire = tag (field_number<<3 | wire_type) + value. Đây là vì sao binary nhỏ ~3× JSON.

3. gRPC vs REST — hiệu năng

    
Kéo số lượng message để thấy chênh lệch wire bytes / thời gian / throughput. Số dựa trên benchmark localhost (README mục 14), tỷ lệ ~3× băng thông, ~5× tốc độ — co giãn theo N.
