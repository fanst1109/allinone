# Nhánh II — Ngữ pháp — Ngữ pháp (Grammar)

Sáu chủ điểm ngữ pháp cốt lõi. Visualization có Web Speech API đọc nguyên câu để vừa nhìn vừa nghe.

## Mục tiêu

- Dùng đúng 12 thì (3 thời × 4 aspect) theo ngữ cảnh.
- Nắm 5 loại câu điều kiện, mệnh đề quan hệ, câu bị động.
- Dùng modal verbs và chuyển lời nói gián tiếp (backshift).

| [Lesson 01](./lesson-01-twelve-tenses/) | Twelve Tenses | 12 thì = 3 thời × 4 aspect, signal words |
| [Lesson 02](./lesson-02-conditionals/) | Conditionals | Zero / first / second / third / mixed |
| [Lesson 03](./lesson-03-relative-clauses/) | Relative Clauses | Defining vs non-defining, lược clause |
| [Lesson 04](./lesson-04-passive-voice/) | Passive Voice | Active↔Passive cho 12 thì + modal |
| [Lesson 05](./lesson-05-modal-verbs/) | Modal Verbs | 9 modal, modal perfect |
| [Lesson 06](./lesson-06-reported-speech/) | Reported Speech | Backshift, reporting verbs |
