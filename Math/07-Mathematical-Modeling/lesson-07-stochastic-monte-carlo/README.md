# Lesson 07 — Mô hình ngẫu nhiên (Monte Carlo, Markov)

> 🚧 **Khung** — folder và mục tiêu đã có; nội dung chi tiết + visualization sẽ bổ sung sau.

## Mục tiêu (dự kiến)

- Khi hệ có **yếu tố ngẫu nhiên**, mô hình hóa bằng xác suất thay vì phương trình tất định.
- **Mô phỏng Monte Carlo**: ước lượng đại lượng khó tính bằng cách lấy mẫu ngẫu nhiên (vd ước lượng π, tính tích phân, rủi ro tài chính); tốc độ hội tụ ~ 1/√N.
- **Xích Markov (Markov chain)**: trạng thái tương lai chỉ phụ thuộc hiện tại; ma trận chuyển; phân phối dừng (stationary).
- Khi nào chọn mô hình ngẫu nhiên thay vì tất định; ưu/nhược điểm.
