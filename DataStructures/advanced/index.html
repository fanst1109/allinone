Nhóm 3 — Nâng cao

4 bài: Graph, Union-Find, Segment Tree & Fenwick, Advanced Structures.

Nhóm này dạy gì?

Đồ thị, range query và các cấu trúc dùng trong hệ thống lớn. Sau nhóm này bạn đã có đủ kiến thức cho phần lớn bài toán phỏng vấn nâng cao và hiểu cách database (B-tree), cache (LSM-tree, Bloom filter), Redis (Skip list) hoạt động bên dưới.

  • Graph: biểu diễn (matrix vs list), 3 thuật toán nền (BFS, DFS, Dijkstra).
  • Union-Find: amortized O(α(n)) ≈ O(1) với union by rank + path compression.
  • Segment Tree / Fenwick: range query + update O(log n), lazy propagation.
  • Advanced: B-tree (DB), Skip list (Redis ZSET), LSM-tree (Cassandra/RocksDB), Bloom filter.
