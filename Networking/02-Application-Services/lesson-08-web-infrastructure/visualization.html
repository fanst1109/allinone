Lesson 08 — Hạ tầng web quy mô lớn

Ba module tương tác: mô phỏng load balancer với nhiều thuật toán, mô phỏng CDN cache hit/miss, và sơ đồ kiến trúc end-to-end với animation request đi qua từng tầng.

1. Mô phỏng cân bằng tải (Load Balancer)

Chọn thuật toán và số backend, gửi từng request hoặc tự động. Đánh dấu 1 backend "chết" để xem health check loại ra.

Nhấn "Gửi request" để bắt đầu...

2. Mô phỏng CDN — cache hit / miss & độ trễ

Chọn vùng người dùng và nội dung. CDN dùng cache hit/miss để quyết định phục vụ từ edge hay về origin. So sánh độ trễ trực quan.

Chọn vùng người dùng và nội dung để xem phân tích...

3. Kiến trúc web quy mô đầy đủ — request đi qua từng tầng

Nhấn từng bước để xem request của người dùng đi qua CDN → Load Balancer → Web Server → Cache → Database, kèm giải thích tại mỗi tầng.

Nhấn "Bước tiếp theo" để theo dõi hành trình của một request HTTPS từ trình duyệt tới database và ngược lại.
