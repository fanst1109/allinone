Lesson 06: P-value, Power & Effect Size

Hiểu đúng p-value, Type I/II error, power curve, Cohen's d, multiple testing

1. Power Curve — Ảnh hưởng của n và Effect Size

Kéo slider n và Cohen's d. Xem power = P(reject H₀ | H₁ đúng) thay đổi. Mục tiêu: power ≥ 0.80 (đường ngang xanh).

2. Type I / Type II Error — Hai phân phối

Phân phối dưới H₀ (trái) và H₁ (phải). Kéo α để thay đổi ngưỡng reject. Thấy trade-off: giảm Type I → tăng Type II.

3. Multiple Testing — Xác suất False Positive tăng theo m

Kéo số lượng test. Ô xanh = correct negative (H₀ đúng, không reject). Ô đỏ = false positive (H₀ đúng nhưng vô tình reject). Tất cả tests dưới đây đều H₀ thật sự đúng.
