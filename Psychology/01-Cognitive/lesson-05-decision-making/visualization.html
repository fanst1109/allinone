Lesson 05: Decision Making

Chọn lottery để ước tính loss aversion của bạn. Xem value function S-shape và probability weighting. Mọi phản hồi dựa trên dữ liệu Prospect Theory thật.

1. Đo Loss Aversion Của Bạn

5 quyết định liên tiếp. Mỗi lần: bạn được mời tham gia một bet. Chọn Chấp nhận hoặc Từ chối.

2. Prospect Theory Value Function (S-shape)

Hàm value v(x) = x^α (gains, concave) và v(x) = −λ(−x)^β (losses, convex). Kéo slider để thay đổi λ.

λ = 2.25
Kahneman & Tversky (1992): α = β ≈ 0.88, λ ≈ 2.25. Hàm concave phần gain (risk-averse) và convex phần loss (risk-seeking). Điểm uốn tại reference point (0).

3. Probability Weighting Function

w(p) = p^γ / [p^γ + (1−p)^γ]^(1/γ). Đường chéo = không bias. Đường cong = overweight nhỏ, underweight lớn.

γ = 0.69
γ = 1 → đường thẳng (không bias). γ < 1 → S-shape đặc trưng. w(0.01) ≈ 4–6% (overweight 4x). w(0.90) ≈ 71% (underweight). Giải thích lottery + insurance paradox.
