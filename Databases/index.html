Databases

Cơ sở dữ liệu — lưu trữ, truy vấn và bảo vệ dữ liệu bền vững, nhất quán, hiệu quả ở quy mô lớn.

Lĩnh vực này dạy gì?

Từ SELECT đầu tiên tới hệ thống phân tán nhiều máy. Giúp trả lời: "vì sao cần DBMS thay vì file?", "index làm truy vấn nhanh lên bằng cách nào?", "transaction đảm bảo tiền không bị mất khi chuyển khoản ra sao?", "khi nào dùng NoSQL thay vì SQL?".

  • 17 bài chia 3 nhóm theo độ khó — đọc tuần tự, nhóm sau dùng kiến thức nhóm trước.
  • Bắt đầu từ nhóm Nền tảng (mô hình quan hệ + SQL).
  • Mỗi lesson có README (lý thuyết + bài tập + lời giải) và visualization.html (minh họa tương tác: chạy thử query, mô phỏng index, transaction…).
  • Liên quan: DataStructures (B-tree, hash table cho index) và Set Theory (mô hình quan hệ).
