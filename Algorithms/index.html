Algorithms

Kỹ thuật & paradigm giải thuật — ~52 lesson chia 8 tier, minh họa bằng Go.

Lĩnh vực này dạy gì?

Các kỹ thuật và paradigm giải quyết vấn đề bằng thuật toán: phân tích độ phức tạp, sắp xếp, tìm kiếm, greedy, quy hoạch động, thuật toán đồ thị, thuật toán chuỗi, và chuyên đề nâng cao.

  • Bổ trợ DataStructures — cấu trúc dạy tổ chức dữ liệu, Algorithms dạy xử lý.
  • Mỗi lesson: README (lý thuyết + code Go inline + walk-through + bài tập có lời giải) + visualization tương tác animate thuật toán.
  • Đọc tuần tự — Tier 0 (Big-O) là nền cho mọi tier sau.
