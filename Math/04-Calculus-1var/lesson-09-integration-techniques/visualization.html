Lesson 09 (T4) — Kỹ thuật tính tích phân

Bốn cách "biến tích phân lạ về tích phân quen". Mọi kết quả đều được kiểm chứng bằng tích phân số (tổng điểm giữa, n=2000) để bạn thấy công thức đúng.

1. Tích phân từng phần — ∫u dv = uv − ∫v du

Chọn một tích phân; xem cách chọn u, dv theo LIATE và nguyên hàm thu được. Cột "số" so nguyên hàm F(b)−F(a) với tích phân số.

2. Đổi biến lượng giác — diện tích dưới cung tròn

∫₀ᵃ √(1−x²) dx tính bằng đặt x = sinθ, ra ½(a·√(1−a²) + arcsin a). Đó chính là diện tích vùng tô dưới cung tròn đơn vị. Tại a=1 → π/4 (¼ hình tròn).

a = 1.00

3. Phân tích thành phân thức đơn giản

Phân thức hữu tỉ tách thành tổng các mảnh A/(x−r), mỗi mảnh tích phân ra A·ln|x−r|.

4. Tích phân suy rộng — tiêu chí p

∫₁^b x^(−p) dx khi b→∞: hội tụ ⇔ p > 1 (giá trị 1/(p−1)); p ≤ 1 phân kỳ. Kéo p và cận trên b để xem tổng tiến tới giới hạn hay vọt lên.

p = 2.0 b = 20
