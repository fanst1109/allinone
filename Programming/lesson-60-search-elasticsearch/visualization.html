Lesson 60 — Search & Elasticsearch

Ba mô-đun tương tác: (1) Inverted index builder — nhập document, build index, query term highlight doc khớp; (2) Analyzer pipeline — animate tokenize → lowercase → stop word → stemming; (3) TF-IDF / BM25 scoring — query + docs, xem điểm relevance tách từng thành phần.

1. Inverted index builder

Mỗi dòng là một document. Bấm Build index để đảo ngược thành term → posting list, rồi gõ một term để tra cứu — chỉ những doc trong posting list được highlight, KHÔNG scan doc nào khác.

2. Analyzer pipeline

Nhập text rồi bấm Phân tích để xem nó chạy qua từng bước. Cùng pipeline này áp cho cả lúc index lẫn lúc search → query "running dogs" khớp được doc chứa "runs".

① Tokenizer — cắt theo khoảng trắng/ký tự
② Lowercase filter — hạ chữ thường
③ Stop word filter — bỏ the/is/a/are…
④ Stemming filter — đưa về gốc từ

3. TF-IDF / BM25 scoring

Query một term trên các document → xem điểm relevance tách thành TF (nhắc nhiều = liên quan) và IDF (term hiếm = quan trọng). Đổi giữa TF-IDF và BM25 (mặc định Elasticsearch).
