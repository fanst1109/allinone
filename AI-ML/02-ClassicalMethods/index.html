Tier 2 — Classical Methods

Tree-based + kernel methods — vẫn dominate tabular data trong industry. KNN, decision tree, random forest, SVM, gradient boosting.

Tier này dạy gì?

  • KNN: instance-based, distance metrics, curse of dimensionality.
  • Decision tree: entropy, information gain, Gini, CART, pruning.
  • Random forest: bagging, OOB error, feature importance.
  • SVM: margin maximization, kernel trick (RBF, polynomial).
  • Gradient boosting: AdaBoost → GBM → XGBoost/LightGBM.
