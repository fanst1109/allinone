Lesson 08 — Op-amp ứng dụng

Chọn loại mạch để khám phá: mạch cộng (summing), mạch trừ (difference), mạch tích phân + comparator, Schmitt trigger. Mỗi mạch có sơ đồ SVG, thanh trượt tương tác và tính toán thời gian thực.

1. Mạch cộng (Summing Amplifier)

V_out = −R_f × (V1/R1 + V2/R2 + V3/R3)
Dấu âm: đầu vào vào chân đảo (−). Virtual ground tại nút summing.

Kéo thanh trượt để thay đổi V_in và điện trở. Sơ đồ mạch cập nhật nhãn; kết quả tính ngay bên dưới.

2. Mạch trừ / Khuếch đại vi sai (Difference Amplifier)

V_out = (R_f / R_in) × (V2 − V1)
Khi R_in = R_f: V_out = V2 − V1
Nhiễu chung V_cm (giống nhau trên cả 2 đầu vào) bị triệt tiêu hoàn toàn.

3. Mạch tích phân (Integrator)

V_out(t) = −(1/RC) × ∫V_in dt
Sóng vuông vào → sóng tam giác ra (tích phân từng đoạn tuyến tính)
Biên độ tam giác = V_in × (T/2) / (R×C)

V_in — Sóng vuông

V_out — Sóng tam giác (tích phân)

4. Comparator (Mạch so sánh)

V_in > V_ref → V_out = +V_sat
V_in < V_ref → V_out = −V_sat
Không hồi tiếp → 1 ngưỡng → có thể rung khi V_in ≈ V_ref
Kéo thanh trượt để xem kết quả...

5. Schmitt Trigger (Comparator có hồi tiếp dương)

V_TH = +V_sat × R2 / (R1 + R2)   (ngưỡng trên)
V_TL = −V_sat × R2 / (R1 + R2)   (ngưỡng dưới)
Hysteresis = V_TH − V_TL = 2 × V_sat × R2/(R1+R2)

Vùng trễ (hysteresis window): thanh ngang 0V ở giữa, vùng xanh = window an toàn, vạch đỏ = V_in hiện tại.
