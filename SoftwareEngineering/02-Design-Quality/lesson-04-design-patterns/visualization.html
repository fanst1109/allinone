Design patterns (mẫu thiết kế GoF)

Ba mô-đun: (1) thư viện pattern — bấm xem vấn đề + sơ đồ + ví dụ; (2) quiz chọn pattern cho tình huống; (3) Strategy demo — đổi thuật toán tính phí ship lúc chạy.

1 · Thư viện pattern — bấm để xem nó giải vấn đề gì

Chọn nhóm GoF rồi bấm một pattern. Mỗi pattern nêu: vấn đề cụ thể → cách giải → ví dụ ngắn.

2 · Chọn pattern cho tình huống

Đọc tình huống, chọn pattern phù hợp nhất. Sai cũng có giải thích.

3 · Strategy demo — đổi thuật toán tính phí ship lúc chạy

Cùng một giỏ hàng, đổi strategy tính phí → kết quả đổi tức thì mà code giỏ hàng KHÔNG đổi.
