Nhóm 3 — Logic & Boolean Algebra

1 bài (đang mở rộng): mệnh đề, đại số Boolean, De Morgan, tautology.

Nhóm này dạy gì?

Logic là ngôn ngữ chính xác để diễn đạt điều kiện, mệnh đề, và tương đương. Sau nhóm này bạn sẽ:

  • Đọc/viết được bảng chân lý (truth table) cho biểu thức bất kỳ.
  • Áp dụng De Morgan, distributive, absorption để rút gọn điều kiện trong code.
  • Phân biệt tautology / contradiction / contingency và biết kiểm tra.
  • Hiểu mối liên hệ chéo: Logic ↔ Bitwise (Nhóm 1) ↔ Set (Nhóm 2) — cùng đại số Boolean trên 3 miền khác nhau.
