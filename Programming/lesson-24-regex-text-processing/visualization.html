Lesson 24 — Regex & Text Processing

3 module dưới đây giúp bạn (1) chơi với regex thật trên trình duyệt, gõ pattern và text → highlight match; (2) build pattern từng bước bằng cách click các "token" (digit, anchor, quantifier...) → ghép thành regex hoàn chỉnh; (3) chọn pattern thực tế (email/phone/URL/date/IP/UUID) và xem giải thích từng phần + test case mẫu.

⚠ Lưu ý: trình duyệt chạy regex JavaScript (ECMAScript), không phải Go RE2. Đại đa số pattern trong lesson tương thích cả hai. Một số khác biệt nhỏ: JS dùng (?<name>...) cho named group, Go dùng (?P<name>...). Backreference và lookahead JS có nhưng Go thì KHÔNG (xem mục 13 của README).

💡 Gợi ý xây pattern:
  • Validate toàn chuỗi là số: ^ + \d + + + $ → ^\d+$
  • Phone VN: ^ + ( + \+84… (gõ thêm trong playground vì builder không có literal) | + 0 + ) + \d + {9} + $
  • Date: ^ + \d{4} + - + \d{2} + - + \d{2} + $
