Tier 1 — Foundations

Pipeline ML end-to-end, regression cơ bản (linear/logistic), bias-variance & cross-validation. Nền cho cả 4 tier sau.

Tier này dạy gì?

  • Pipeline ML: data → feature → model → loss → train → evaluate.
  • Linear regression: closed-form (normal equation), gradient descent, ridge/lasso.
  • Logistic regression: sigmoid, cross-entropy; multi-class softmax; calibration + ROC/AUC.
  • Bias-variance trade-off, regularization (L1/L2, dropout, early stop), cross-validation.
