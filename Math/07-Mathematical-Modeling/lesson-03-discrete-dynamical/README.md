# Lesson 03 — Mô hình rời rạc (Phương trình sai phân)

> 🚧 **Khung** — folder và mục tiêu đã có; nội dung chi tiết + visualization sẽ bổ sung sau.

## Mục tiêu (dự kiến)

- **Phương trình sai phân** (difference equation): mô hình khi thời gian đi theo *bước rời rạc* (năm, tháng, thế hệ): xₙ₊₁ = f(xₙ).
- Ví dụ: **lãi kép** (xₙ₊₁ = (1+r)·xₙ), **dân số rời rạc**, mô hình logistic rời rạc xₙ₊₁ = r·xₙ·(1−xₙ).
- **Điểm cân bằng (fixed point)** và **ổn định**: khi nào dãy hội tụ, dao động, hay hỗn loạn (chaos).
- So sánh mô hình rời rạc ↔ liên tục (ODE ở [Lesson 04](../lesson-04-continuous-ode-models/)): khi nào dùng cái nào.
