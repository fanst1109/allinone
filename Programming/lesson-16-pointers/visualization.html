1. Pointer Playground — memory cell live

Hai biến x và p. x là int, p là *int trỏ tới x (hoặc nil). Bấm các nút để mô phỏng:

x chưa khởi tạo. Bấm "x = 42" để bắt đầu.
Quan sát: p chỉ chứa địa chỉ (8 byte), không lưu giá trị. Khi gán *p = 99, ô memory của x đổi giá trị — bạn thấy cả 2 cell cùng cập nhật. Khi p = nil, mũi tên biến mất; bấm "đọc *p" lúc đó → panic giả lập.
