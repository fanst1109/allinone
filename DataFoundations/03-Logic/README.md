# Nhóm 3 — Logic & Boolean — Logic mệnh đề & đại số Boolean

Logic mệnh đề và đại số Boolean — ngôn ngữ chính xác cho mọi điều kiện trong code, và mối liên hệ chéo Logic ↔ Bitwise ↔ Set.

## Mục tiêu

- Đọc / viết truth table cho các phép logic.
- Áp dụng đại số Boolean (De Morgan, absorption) để rút gọn điều kiện.
- Kiểm tra tautology và lập luận hình thức.
