2. Closure Explorer — hàm "nhớ" biến

  
makeAdder(x) trả về một hàm con bắt giữ x. Tạo vài adder với x khác nhau, rồi gọi chúng để thấy mỗi closure tra x từ environment captured của riêng nó.

  

    
fn makeAdder(x) {
  fn inner(n) { return n + x; }   // x là biến tự do, bắt từ makeAdder
  return inner;
}

    

      
      
      
      
    

    

    

      

      

        
        
        
        
        
        
      

      

    

  


  
  
3. Closure Loop Bug — bắt biến, không bắt giá trị

  
Vòng lặp tạo nhiều closure. Nếu cả vòng dùng một biến lặp chung (bug), mọi closure trỏ cùng ô nhớ → in cùng giá trị cuối. Bật/tắt để so sánh với bản đã sửa (biến mới mỗi vòng).
