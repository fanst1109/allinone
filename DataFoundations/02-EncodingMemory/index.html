Nhóm 2 — Encoding & Memory

Cách máy lưu data tổng quát: text (UTF-8), byte order, hash function.

Nhóm này dạy gì?

  • Character encoding: ASCII → Unicode → UTF-8 (1-4 byte variable). Vì sao "café" lưu 5 byte chứ không 4.
  • Endianness: big vs little endian; vì sao network byte order là big endian; alignment + struct padding.
  • Hashing: tính chất (uniform, deterministic), collision handling (chaining vs open addressing), load factor, birthday paradox.
