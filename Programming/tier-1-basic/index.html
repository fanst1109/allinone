Tier 1 — Go Basic

12 lesson về cú pháp Go cơ bản, kết thúc bằng mini-project CLI todo.

Tier này dạy gì?

Tier 1 dạy cú pháp Go từ hello world đến struct: biến, hàm, slice, map, string, struct, pointer. Sau Tier 1, bạn có thể đọc và viết được code Go cỡ ~200 dòng, hiểu mọi thứ trong fmt, strings, strconv. Lesson 17 là mini-project CLI todo — bài tổng kết cả tier.

Tiền đề: đã xong Tier 0 (đặc biệt L05 — đã chạy được hello world).
