Tier 3 — Go Advanced

12 lesson đi sâu vào Go: generics, reflect, memory/GC, profiling, concurrency patterns, design patterns.

Tier này dạy gì?

Tier 3 là phần "đi sâu" của Go: generics (Go 1.18+), reflect runtime, escape analysis & GC tri-color, pprof profiling, benchmark đúng cách, các pattern concurrency thực chiến, và idioms thiết kế Go. Kết thúc bằng mini-project web scraper concurrent có rate limit + context cancel.

Tiền đề: đã xong Tier 2 (đặc biệt L27 channel, L29 context).
