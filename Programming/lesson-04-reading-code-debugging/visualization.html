Lesson 04 — Đọc code & Debug

Ba module tương tác: trace happy path qua codebase, đọc stack trace, và chơi git bisect.

1. Codebase Explorer — trace happy path

Bạn vừa clone payment-api. Click vào từng file để xem nội dung. Click tên hàm gạch chân (giả lập Go to Definition = F12) để nhảy tới định nghĩa. Hoặc bấm Auto trace để xem cách top-down đi từ entry point đến điểm bug.

3. Bisect simulator — chơi git bisect

20 commit từ v1.0.0 (good) đến HEAD (bad). Bug introduce ở 1 commit ngẫu nhiên giữa. Git binary search — bạn chỉ cần đánh dấu mỗi commit Git suggest là good hay bad.
