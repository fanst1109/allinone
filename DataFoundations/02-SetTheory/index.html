Nhóm 2 — Lý thuyết tập hợp

1 bài (đang mở rộng): tập hợp, ánh xạ, quan hệ tương đương.

Nhóm này dạy gì?

Lý thuyết tập hợp là ngôn ngữ chung của toán rời rạc và cấu trúc dữ liệu. Sau nhóm này bạn sẽ thấy:

  • HashSet chính là cài đặt thực tế của tập hợp toán học.
  • HashMap = ánh xạ (function/mapping) từ key → value.
  • Union-Find = quan hệ tương đương (equivalence relation) — chia tập thành các lớp tương đương.
  • Graph = quan hệ hai ngôi trên tập đỉnh.
