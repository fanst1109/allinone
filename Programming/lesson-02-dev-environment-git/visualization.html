2. Branch Tree — Fast-Forward vs 3-Way Merge

Chọn kịch bản, ấn "Bắt đầu" rồi click các nút commit/merge để xem cây branch tiến triển. Quan sát: khi nào Git tạo merge commit (M), khi nào chỉ tua thẳng.

Bắt đầu kịch bản A: ấn "commit on main" 2 lần, "checkout -b feature", "commit on feature" 2 lần, rồi "merge". Vì main không tiến thêm sau khi feature rẽ → Git fast-forward (không có merge commit).
