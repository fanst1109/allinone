Tier 2 — Go Intermediate

12 lesson: interface, error, IO, JSON, testing, goroutine, channel, sync, context.

Tier này dạy gì?

Sau khi nắm syntax Go ở Tier 1, Tier 2 đi vào những thứ khiến Go là Go: interface implicit, error là value, goroutine và channel, context. Cũng chuẩn hoá kỹ năng làm việc: packages/modules, file I/O, JSON, regex, time, testing.

Tiền đề: đã xong Tier 1, đã viết được mini-project CLI todo.
