AI-ML

Học máy & Trí tuệ nhân tạo ứng dụng — 8 bài, từ linear regression cổ điển tới RAG và CLIP multimodal.

Lĩnh vực này dạy gì?

Ráp toán nền (Vectors, Math, Statistics) thành các thuật toán ML thật. Mỗi bài có viz tương tác minh hoạ pipeline / model / loss.

  • Pipeline ML end-to-end: data → feature → model → loss → train → evaluate.
  • Linear & logistic regression — ML cổ điển, nền cho mọi model phức tạp hơn.
  • Neural network — forward + backprop bằng tay (chain rule).
  • Word embedding, vector DB, RAG.
  • CLIP — multimodal embedding (ảnh + text cùng không gian).
