# Tier 4 — Applied (Chuyên đề ứng dụng)

5 lesson ứng dụng các nguyên lý vi mô / vĩ mô vào các chuyên đề thực tế.

## Mục tiêu

- Hiểu lý thuyết **thương mại quốc tế** và tỉ giá hối đoái.
- Áp dụng **behavioral economics** vào quyết định cá nhân + thiết kế nudge.
- Phân tích **development economics** — vì sao nước giàu/nghèo + RCT method.
- Bước đầu vào **econometrics** — hồi quy, IV, DiD.
- Hiểu **financial economics** — CAPM, EMH, diversification.
