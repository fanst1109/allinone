# Nhóm 2 — Trung cấp — Cây và các biến thể

Cây và các biến thể là backbone của database index, autocomplete, scheduler. Tiền đề: hoàn thành Nhóm 1 (cần Big-O, Array, Linked List).

## Mục tiêu

- Hiểu thuật ngữ cây và 3 cách duyệt (pre/in/post-order).
- Cài đặt BST và biết khi nào cần cây cân bằng (AVL, Red-Black).
- Dùng heap cho priority queue và trie cho autocomplete / prefix search.

## Lộ trình 5 bài

| Bài | Chủ đề | Khái niệm chính |

| [Lesson 01](./lesson-01-tree/) | Tree | Thuật ngữ, binary tree, các cách duyệt |
| [Lesson 02](./lesson-02-binary-search-tree/) | Binary Search Tree | BST, search / insert / delete |

- **visualization.html**: trang tương tác, mở trực tiếp trong trình duyệt là chạy. Có nút **📖 Đọc README** để xem lý thuyết song song.
