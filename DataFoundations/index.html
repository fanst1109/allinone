DataFoundations

Nền tảng cho Data Structures — binary, hex, bitwise, set theory.

Lĩnh vực này dạy gì?

Những kiến thức nền cần có trước hoặc song song với DataStructures. Trả lời các câu hỏi như: "vì sao int64 chỉ tới ~9.2·10¹⁸?", "bitmask DP là gì?", "HashMap có liên quan gì tới ánh xạ?".

  • Hiểu cách máy tính biểu diễn số (binary, hex, two's complement).
  • Thao tác trực tiếp trên bit — bitmask, popcount, các trick kinh điển.
  • Lý thuyết tập hợp — nền cho HashSet, HashMap, Union-Find, Graph.
