Tier 8 — Capstone Project

3 lesson xây 1 dịch vụ thật từ đầu: URL Shortener có Analytics — design → implement → deploy & observe.

Tier này dạy gì?

Capstone gắn kết toàn bộ lộ trình vào một dự án production-grade duy nhất: URL Shortener có Analytics (như bit.ly). Bạn sẽ đi từ design doc → implement đầy đủ (REST API + Postgres + Redis cache + async click queue + tests) → dockerize + deploy Kubernetes (kind) + thêm observability (log/metrics/trace) + load test.

  • Tích hợp Tier 4 (REST), Tier 5 (DB + cache), Tier 6 (queue + event), Tier 7 (observability + Docker + K8s).
  • Đầu ra: 1 portfolio piece thật, chạy được, có thể demo cho nhà tuyển dụng.

Tiền đề: đã xong Tier 7 (toàn bộ kiến thức production).
