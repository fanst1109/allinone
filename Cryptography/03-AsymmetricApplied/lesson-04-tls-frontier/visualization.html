2. Forward Secrecy (PFS)

❌ Không có PFS (RSA Key Exchange)

Client mã hóa pre-master-secret bằng server pubkey.
Attacker lưu traffic hôm nay.
2 năm sau server private key bị lộ
→ Giải mã pre-master-secret
→ Derive session key
→ Decrypt toàn bộ past traffic!
✓ Có PFS (ECDHE)

Session key = ECDHE(a, b): a, b ephemeral.
a, b bị XÓA ngay sau handshake.
Attacker lưu traffic hôm nay.
Server private key bị lộ sau này
→ Chỉ verify cert — không liên quan session key
→ Past sessions vẫn an toàn!
TLS 1.3 loại bỏ hoàn toàn RSA key exchange — mọi session đều dùng ECDHE → PFS là bắt buộc.

3. Post-Quantum Cryptography

⚠ Shor's Algorithm (1994): Máy tính lượng tử factor số nguyên trong O((log n)³) — polynomial! RSA-2048, DH, ECDH, ECDSA đều bị phá. Timeline CRQC ước tính ~2030-2035.

4. Schnorr Zero-Knowledge Proof

Prove "Tôi biết x sao cho y = g^x mod p" mà không tiết lộ x. Protocol 3 messages: Commit → Challenge → Response.
