Nhóm 3 — Math Foundations

Toán logic cho data: lý thuyết tập hợp, đại số Boolean / logic mệnh đề.

Nhóm này dạy gì?

  • Set theory: tập hợp, phép toán (∪, ∩, \, △), ánh xạ, quan hệ tương đương — nền cho mô hình quan hệ (relational), HashSet, Union-Find.
  • Boolean logic: truth table, đại số Boolean, De Morgan, tautology — nền cho mọi predicate trong code, formal reasoning, circuit minimization.

Đây là toán cơ bản đủ cho ứng dụng data. Phần sâu hơn (combinatorics, cardinality vô hạn, predicate logic ∀∃, proof techniques) ở Math/05.
