Lesson 06 — Hello World & Toolchain

Ba module tương tác để quen tay với toolchain Go: (1) mô phỏng cấu trúc project khi chạy go mod init/tidy/build, (2) chọn GOOS + GOARCH xem binary ra sao, (3) click vào từng dạng import để hiểu ý nghĩa.

1. Project Structure Simulator

Đi qua 5 bước từ folder trống đến binary chạy được. Click Bước tiếp → để chạy lần lượt từng lệnh, xem tree thay đổi. Click vào file trong tree để xem nội dung.

📁 ~/myapp/
Click vào file bên trái để xem nội dung
(chưa chọn file)
Bắt đầu: folder ~/myapp/ rỗng. Click Bước tiếp để chạy go mod init myapp.

3. Import Explorer

5 dạng import trong Go. Click vào từng dòng để xem giải thích.

package main import "fmt" // (1) single import ( // (2) group "os" "strings" "github.com/google/uuid" ) import ( // (3) alias cryptorand "crypto/rand" mathrand "math/rand" ) import _ "github.com/lib/pq" // (4) blank import . "fmt" // (5) dot — tránh!
Click một dòng import bên trên để xem giải thích.
