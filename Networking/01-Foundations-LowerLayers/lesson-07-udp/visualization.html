Lesson 07 — Tầng giao vận: UDP

Công cụ tương tác: Bóc tách UDP header (click từng trường), mô phỏng gửi datagram (mất gói / đảo thứ tự), bảng tra cổng phổ biến, minh họa demultiplexing nhiều tiến trình.

1. Bóc tách UDP Header

Click vào từng trường trong sơ đồ để xem giải thích chi tiết. Hoặc nhập 8 byte hex phía dưới để phân tích gói thực tế.

Source Port 16 bit 5000
Destination Port 16 bit 53
Length 16 bit 28
Checksum 16 bit 0xB24A
Payload (Data) — 20 byte DNS query
