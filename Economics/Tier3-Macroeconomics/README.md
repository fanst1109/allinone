# Tier 3 — Macroeconomics

Vĩ mô: nhìn toàn bộ nền kinh tế. Sau khi nắm vi mô (Tier 2), Tier 3 đặt câu hỏi: *làm thế nào tổng hợp các quyết định cá nhân thành GDP, tăng trưởng, lạm phát, thất nghiệp*?

## Mục tiêu

- Đo GDP, phân biệt Nominal/Real, hiểu Deflator + CPI.
- Phân tích tăng trưởng dài hạn qua Solow model.
- Mô hình cân bằng ngắn hạn qua IS-LM và AD-AS.
- Áp dụng Taylor rule + multiplier cho chính sách tiền tệ và tài khóa.
- Hiểu Phillips curve và trade-off ngắn hạn giữa lạm phát + thất nghiệp.
