Command Line Mastery — 3 module tương tác

Module 1: build pipe chain trên log giả lập · Module 2: quiz find/grep · Module 3: visualize permission rwx/octal.

Module 1 — Pipe Builder

Chọn lệnh từ dropdown, "Thêm" để build chain. Mỗi lệnh chạy trên output của lệnh trước. Click "Chạy" để xem từng bước.

Preset:
Pipe đang build:
cat dataset.txt
Output (mỗi giai đoạn hiển thị tối đa 10 dòng đầu):
(click Chạy để xem)

Module 2 — find / grep Quiz

Đọc đề, gõ lệnh vào input. Click "Kiểm tra" — viz so sánh các thành phần lệnh của bạn với mẫu (so khớp linh hoạt: thứ tự flag không quan trọng, dấu ngoặc tuỳ chọn).

Module 3 — Permission Visualizer (chmod)

Click vào ô r/w/x để bật/tắt. Hệ thống tự tính octal + giải thích ai có quyền gì.

Hoặc nhập trực tiếp:
Preset:
Octal: 000
Symbolic: ---------
chmod: chmod 000 file
Chưa có quyền nào.
Quy tắc tính: r=4, w=2, x=1 — cộng từng nhóm

Mỗi nhóm (u/g/o) có 3 bit. r=4, w=2, x=1.
Vd rwxr-x---: u = 4+2+1 = 7, g = 4+0+1 = 5, o = 0 → 750.
