3. Demo sai số float — vì sao 0.1 + 0.2 ≠ 0.3?

Trong máy tính, float64 lưu số dưới dạng nhị phân với 52 bit định trị. 0.1₁₀ trong hệ 2 là chuỗi tuần hoàn vô hạn 0.000110011001100…₂, phải cắt sau 52 bit → lệch một chút. Khi cộng, sai số tích lũy.

Bấm các nút trên để xem demo. Mỗi thí nghiệm in cả phép `==` trực tiếp và `almostEqual` với epsilon.

Tự thử phép tính

Cách so sánh đúng: almostEqual

func almostEqual(a, b, eps float64) bool {
    diff := a - b
    if diff < 0 { diff = -diff }
    return diff < eps
}

// Dùng:
almostEqual(0.1+0.2, 0.3, 1e-9)  // true ✓
