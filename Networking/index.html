Networking

Mạng máy tính — 3 tầng × 8 bài, từ bit trên dây tới request HTTP và hạ tầng web quy mô lớn.

Triết lý

  • Thấy được trước, đào sâu sau: từ thứ quen thuộc (gõ một địa chỉ web) rồi bóc từng tầng.
  • Cụ thể bằng số thật: subnet tính từng bit, cửa sổ TCP tính ra byte, độ trễ tính ra ms.
  • Trực giác đời thường: gói tin ↔ lá thư, router ↔ bưu cục, port ↔ số phòng, TCP ↔ cuộc gọi xác nhận.
  • Visualization tương tác: tự chia subnet, xem bắt tay TCP, lần theo gói qua router, bóc khung Ethernet.
