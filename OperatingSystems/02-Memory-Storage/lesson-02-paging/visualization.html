Paging — Phân trang bộ nhớ

Module 1 — Bộ dịch địa chỉ (Address Translator)

Nhập địa chỉ ảo và bảng trang. Xem từng bước tách page/offset và tính địa chỉ vật lý.

Bảng trang (page number → frame number). Thêm trang bằng nút bên dưới.

Phân tích bit:

Page number bits Offset bits

Module 2 — Bản đồ bộ nhớ (Logical → Physical)

Xem bố cục trang logic và frame vật lý. Frame màu vàng = trang đang được dịch; xanh lá = frame đích.

Module 3 — Phân mảnh trong (Internal Fragmentation)

Nhập kích thước tiến trình và page size để tính số trang cần thiết và phân mảnh trong.
