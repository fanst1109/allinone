Programming

Học lập trình bằng Go từ tư duy đến production — 79 lesson chia 9 tier.

Lĩnh vực này dạy gì?

Lộ trình tự học từ tư duy lập trình thuần (chưa chạm code) đến xây dựng và deploy dịch vụ production trên Kubernetes có đầy đủ logging / metrics / tracing. Ngôn ngữ chính là Go (Golang).

  • 79 lesson chia 9 tier — đọc tuần tự, tier sau dùng kiến thức tier trước.
  • 6 mini-project + 1 capstone — gắn kết kiến thức thành sản phẩm chạy thật.
  • Mỗi lesson: README chi tiết, solutions.go chạy được, visualization.html tương tác.
  • Tiền đề: không bắt buộc. Có ích nếu đã đọc DataFoundations.
Bắt đầu từ đâu?
Tier 0
Foundation / Mindset
5 lesson nền tảng: tư duy lập trình, môi trường dev, Git, command line, đọc code. Không Go-specific, áp dụng cho mọi ngôn ngữ.
▶ Vào Tier 0 📖 README
Tier 1
Go Basic
12 lesson Go cơ bản: hello world, biến, hàm, slice, map, struct, pointer. Kết thúc với mini-project CLI todo.
▶ Vào Tier 1
Tier 2
Go Intermediate
12 lesson: interface, error, IO, JSON, regex, time, testing, goroutine, channel, sync, context.
▶ Vào Tier 2
Tier 3
Go Advanced
12 lesson: generics, reflect, unsafe, memory/GC, pprof, benchmark, concurrency patterns, design patterns. Kết thúc với mini-project web scraper concurrent.
▶ Vào Tier 3
Tier 4
Web & Backend
12 lesson: net/http, REST design, framework, validation, JWT auth, TLS, WebSocket, gRPC, GraphQL, graceful shutdown, rate limit. Kết thúc với mini-project REST API đầy đủ.
▶ Vào Tier 4
Tier 5
Database & Storage
8 lesson: database/sql, ORM, transaction, migration, Redis cache, MongoDB, Elasticsearch. Kết thúc với mini-project blog backend.
▶ Vào Tier 5
Tier 6
Distributed & Microservices
10 lesson: CAP, service discovery, MQ (Kafka/NATS), event-driven, saga, CQRS, Raft, microservice patterns, service mesh. Kết thúc với mini-project 3 service.
▶ Vào Tier 6
Tier 7
Production / DevOps / SWE
10 lesson: structured log, Prometheus, OpenTelemetry, Docker, Kubernetes, CI/CD, config, clean architecture, code review, postmortem.
▶ Vào Tier 7
Tier 8
Capstone Project
3 lesson xây 1 dịch vụ thật từ đầu — design doc → implement → deploy có observability. Đầu ra: portfolio piece thực sự.
▶ Vào Tier 8
Tài liệu viết bằng tiếng Việt. Code và lệnh giữ tiếng Anh. Mọi visualization.html mở trực tiếp trong trình duyệt là chạy.
